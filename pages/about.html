Sobre mim

Olá, sou Eduardo Moreira, um entusiasta da programação com experiência em C# e .NET Core. Desenvolvo aplicações utilizando Lógica de Programação, Orientação a Objetos, Arquitetura em Camadas e Testes Automatizados com MSTest.

Tenho conhecimentos sólidos no uso do Entity Framework Core para interação com bancos de dados, especialmente o Microsoft SQL Server. Além disso, minha experiência se estende ao desenvolvimento web com ASP.NET Web APIs, HTML e CSS.

Exploro o front-end com TypeScript e Angular, criando interfaces modernas e intuitivas.
